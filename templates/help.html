Help

Getting Started

Welcome to Workout Manager! This application helps you organize your workout programs, exercises, and workout blocks.

Programs

  • • View Programs: Click on "Programs" in the navigation bar to see all your workout programs.
  • • Add Program: Click the "+" button in the Programs page to create a new program.
  • • Edit Program: Click the edit icon (pencil) next to any program to change its name.
  • • Delete Program: Click the trash icon next to a program to remove it. This will also delete all associated workout blocks.
  • • View Program Details: Click on a program name to view its workout blocks.

Exercises

  • • View Exercises: Click on "Exercises" in the navigation bar to see all available exercises.
  • • Add Exercise: Click the "+" button in the Exercises page to add a new exercise.
  • • Edit Exercise: Click the edit icon (pencil) next to any exercise to change its name.
  • • Delete Exercise: Click the trash icon next to an exercise to remove it.
  • • Exercise Rules: Exercise names must be 50 characters or less and can only contain letters, spaces, and dashes.

Workout Blocks

  • • Add Workout Block: When viewing a program, click "Add Workout Block" to create a new block.
  • • Edit Block: Click the edit icon (pencil) next to any workout block to change its name.
  • • Delete Block: Click the trash icon next to a workout block to remove it.
  • • Remove Program: On the program detail page, click "Remove Program" to delete the entire program and all its blocks.

Tips

  • • Use descriptive names for your programs and exercises to keep them organized.
  • • All delete actions require confirmation and cannot be undone.
  • • You can edit names directly from the list pages without navigating away.
  • • Use the navigation bar at the top to quickly move between sections.
